# Pokémon Data Cleaning

Cleans messy Pokémon DataFrame by fixing inconsistent typing, fixing names, dropping invalid rows, and removing duplicates.

Skills Practiced
1. .str.strip()
2. .str.lower()
3. .str.capitalize()
4. Dropping duplicates with 'subset' parameter
5. Replacing invalid entries with 'df.replace()'
6. Removing blank strings and 'None'
7. Filtering rows with '.dropna()'
